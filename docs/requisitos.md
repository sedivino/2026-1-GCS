# Requisitos do Sistema

## Funcionais
- RF01: O sistema deve permitir adicionar tarefas
- RF02: O sistema deve permitir listar tarefas
- RF03: O sistema deve permitir marcar tarefas como concluídas
- RF04: O sistema deve permitir remover tarefas

## Não Funcionais
- RNF01: Dados persistidos em arquivo JSON
- RNF02: Interface via linha de comando (CLI)
